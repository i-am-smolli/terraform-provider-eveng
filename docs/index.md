# EVE-NG Provider

The EVE-NG provider is used to interact with the EVE-NG (Emulated Virtual Environment Next Generation) API. It allows network engineers and infrastructure practitioners to manage EVE-NG topologies, folders, labs, nodes (virtual machines / virtualized network devices), networks (bridges/clouds), and links as code.

## Example Usage

```terraform
terraform {
  required_providers {
    eveng = {
      source  = "i-am-smolli/eveng"
      version = "~> 0.1.10"
    }
  }
}

provider "eveng" {
  host     = "http://localhost"
  username = "admin"
  password = "eve"
  # insecure = true  # Disable TLS certificate verification (default: false)
}
```
